**Descrição**

É um sistema focado em facilitar o trabalho de uma Pizzaria,

tendo Home, Banner, Introdução, Cardápio e Rodapé. Sua interface

deve ser bem intuitiva e criativa com um Carrossel de imagens das

pizzas feitas pelo pizzaiolo.
